---- MapleOrigin MySQL Database ----

Just execute db_create.sql and db_create_bit.sql and it will create everything needed to run MapleOrigin. db_create_bit
is used for MapleBit website. You will want to change your information in bit_properties and bit_vote

To wipe a server run the wipe.sql. It will keep GM characters and give player's a rewardpoint on their
most voted account with more than 3 votepoints
